# 🤖 NEXUS Bot v5.0

## Umgebungsvariablen (Railway)
| Variable | Beschreibung |
|---|---|
| `DISCORD_TOKEN` | Bot-Token vom Developer Portal |
| `ANTHROPIC_API_KEY` | Claude API Key (für KI-Features) |
| `OWNER_ID` | Deine Discord User ID (Rechtsklick → ID kopieren) |

## Owner-Befehle
- `/wartung-an` — Wartungsmodus AN (Bot ignoriert alle außer Owner)
- `/wartung-aus` — Wartungsmodus AUS
- `/bot-neustart` — Bot neu starten
- `/bot-status` — Vollständiger System-Status
- `/bot-ankuendigung` — Nachricht an alle Server
- `/ki-programmiere` — Bot schreibt sich selbst neuen Code
- `/ki-reset` — KI-Gedächtnis zurücksetzen
